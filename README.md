# MLIP_vib_benchmarking_project
Calculation of vibrational Density of States (vDOS) spectra and free energies for Ab Initio Molecular Dynamics and the most common Machine-Learning Foundation Models

A wrapper to the ASE library can be found in the folder ASE wrapper. There, the script dynamic.py acts as an interface to all benchmarked uMLIPs (excluding the GFN methods).
